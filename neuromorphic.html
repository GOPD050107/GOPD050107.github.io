Neuromorphic Spiking Algorithms

Python simulations bridging circuit hardware with biological sensing efficiency.

The Concept

Traditional computing relies on always-open circuits, which consume vast amounts of power. Neuromorphic engineering mimics biological brains by modulating voltage through discrete energy spikes, dramatically increasing power efficiency. I built Python simulations to model two foundational neuromorphic concepts.

1. Leaky Integrate-and-Fire (LIF) Model

I simulated the electrical behavior of a single biological neuron's membrane. Using Euler integration, the algorithm accumulates a simulated input current. If unstimulated, the voltage "leaks" back to a resting state (-60mV). When the voltage hits a defined threshold (-50mV), the circuit fires an energy spike and enters a 5ms refractory period, rejecting incoming current to reset.

2. Spike-Timing-Dependent Plasticity (STDP)

Moving from a single neuron to a network, I modeled STDP—a biological learning rule for artificial synapses. The algorithm tracks the exact firing timing of pre-synaptic and post-synaptic simulated neurons. Using exponential decay traces, the algorithm automatically adjusts "weight" (connection strength). If the input fires right before the output, the circuit strengthens the pathway; if the timing is reversed, it penalizes it.
