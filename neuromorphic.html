Neuromorphic Spiking Algorithms

Modeling biological neuron dynamics for power-efficient hardware.

The Concept

Conventional circuits are "always-on," leading to high power consumption. Neuromorphic systems increase efficiency by using energy spikes, only transmitting power when a specific threshold is met, similar to biological neurons.

1. Leaky Integrate-and-Fire (LIF)

I architected simulations to model membrane voltage accumulation. The system integrates incoming current, but "leaks" voltage over time if no stimulus is present. Once the threshold is reached, the system fires a spike and enters a refractory period to recharge.

Fig 1: Simulated membrane voltage showing accumulation, threshold firing, and refractory reset.

2. Spike-Timing-Dependent Plasticity (STDP)

This algorithm mimics biological learning. If an input spike occurs just before an output spike, the connection strengthens. If the timing is reversed, the connection weakens. This allows the system to optimize data pathways dynamically.
